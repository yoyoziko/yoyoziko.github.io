Studying devops courses

Situation:

Driven by a desire to enhance my skills in DevOps Engineering, I recently enrolled in an online course focusing on Kubernetes, Jenkins, and other DevOps-related tools. This course aimed to equip me with the knowledge and expertise to effectively develop and manage IT systems.

Task:

    My primary objective was to learn about:

  • Utilizing Kubernetes to efficiently manage application clusters.
  • Implementing Jenkins to create Continuous Integration/Continuous Deployment pipelines.
  • Leveraging cloud services to optimize development and delivery processes.

Action:

Throughout the DevOps Engineer course, I delved into:

  • Installing and configuring Kubernetes.
  • Creating and managing Docker containers for application development and testing.
  • Utilizing Jenkins to establish Continuous Integration pipelines, automating and streamlining software testing and delivery.
  • Exploring cloud services, focusing on creating and managing virtual machines and utilizing Linux operating systems in a cloud environment.

    • Result:

    Upon completing the DevOps Engineer course, I gained a comprehensive understanding of Kubernetes, Jenkins, and other DevOps tools. I am now proficient in:

  • Designing and managing scalable application architectures on the cloud.
  • Implementing Jenkins to automate the software delivery process.
  • Utilizing cloud services and Linux systems effectively.
